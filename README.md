Week 2 – Sales Data Analysis


This project is part of my internship program and focuses on analyzing sales data using Python and Pandas to extract meaningful business insights.

Project Objectives


Create a sales dataset with at least 50 records

Calculate total sales amount for each order

Analyze overall revenue and total orders

Perform product-wise and category-wise sales analysis

Perform monthly sales trend analysis

Export the final dataset as a CSV file


Key Features


Generated a structured sales dataset with order details, product, category, quantity, and price

Calculated total sales using quantity × price

Identified top-performing products and categories

Analyzed monthly sales patterns to identify peak periods

Wrote a business summary highlighting key findings

Exported the final processed data for reporting


Tools Used


Python
Pandas
Google Colab


Files Included


Week2_Sales_Data_Analysis.ipynb – Complete analysis notebook

week2_sales_analysis_output.csv – Final processed dataset


Conclusion


This project demonstrates a complete sales data analysis workflow, from dataset creation to business insight generation, using Python in a real-world internship scenario.
